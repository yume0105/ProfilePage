近業抑制システム

Overview

小児の近視人口の増加は深刻な社会問題となっています。本プロジェクトでは、近業作業（本を読む、タブレットを見るなど）中の姿勢や距離を計測し、適切な距離を保つようユーザーにフィードバックを行うウェアラブルデバイスを開発しました。

Technical Details

Hardware: M5StickC Plus, ToF Sensor (Distance), IMU (Posture)
Software: Arduino IDE (C++), Python (Data Analysis)
Mechanism: メガネに装着可能な小型アタッチメントとして設計。3Dプリンタで筐体を作成。

My Role

プロジェクトリーダーとして、コンセプト設計からハードウェアの実装、被験者実験の設計までを担当しました。特に、子供が装着しても違和感のない軽量化設計に注力しました。
